Darber hinaus gelten Eintrge, fr die ein Betrag beansprucht wird, deren Merkmale aber nicht vorhanden sind, als fehlend. Diese Eintrge entfallen.

Wir stellen fest, dass es Versicherungsnehmer mit mehr als einem Anspruch gibt. Fr diese Versicherungsnehmer rechnen wir die Hhe ihrer Ansprche zusammen. Auf diese Weise werden mehrere Eintrge zu diesen Versicherungsnehmern durch einen Eintrag ersetzt.

Feature-Engineering

Entscheidungsbume treffen fundierte Entscheidungen zur Schichtung der Daten auf der Grundlage der durch verschiedene Merkmale gewonnenen Informationen. Daher werden wir den Prozess der Merkmalsauswahl einschrnken und ihn zur Optimierung dem Modell berlassen. Darber hinaus sehen wir davon ab, die Forderungsbetrge umzuwandeln. Dadurch kann das Modell die Auswirkung von Ausreiern bercksichtigen.

Wir betrachten die folgenden Feature-Engineering-Schritte:

  1. Sinkende Dichte und Region (die Anspruchshhe ist gleichmig auf die Regionen verteilt und die in der Dichte enthaltenen Informationen sind hchstwahrscheinlich in der Flchenvariablen enthalten)
  2. Fassen Sie Marken in zwei Kategorien mit hohen und niedrigen Anspruchszahlen zusammen
  3. Konvertieren der kategorialen Merkmale in numerische.

## RMSE
def model_performance(X_train, X_test, y_train, y_test, model):
    train_pred = model.predict(X_train)
    rmse_train = mean_squared_error(y_train, train_pred, multioutput = 'raw_values')**(1/2)
    print('Training RMSE: ' + str(rmse_train))
    y_pred = model.predict(X_test)
    rmse_test = mean_squared_error(y_test, y_pred, multioutput = 'raw_values')**(1/2)
    print('Test RMSE: ' + str(rmse_test))

    '''ym = y_pred[:,0].mean() - y_test['ClaimAmount'].mean()
    yv = y_pred[:,0].std()**2
    yy = ym**2 + yv
    yx = ((y_pred[:,0] - y_test['ClaimAmount'])**2).mean()
    print(yy)
    print(yx)'''
    if len(pd.DataFrame(y_train).columns) == 2:
        out_pred = (y_pred[:,0]/y_pred[:,1]).mean()
        out_test = (np.array(y_test['ClaimAmount'])/np.array(y_test['Exposure'])).mean()
        print(' Pred - Expected Premium ' + str(out_pred))
        print(' True - Expected Premium ' + str(out_test))
        plots(X.columns, model, y_pred[:,0], y_test['ClaimAmount'], y_pred[:,1], y_test['Exposure'])
    else:
        plots(X.columns, model, y_pred, y_test, y_pred, y_test)
        out_pred = y_pred
        out_test = y_test
    return rmse_train, rmse_test, out_pred, out_test
